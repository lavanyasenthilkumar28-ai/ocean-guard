
# OceanGuard 

A real-time IoT-based marine rescue alert and monitoring system designed to improve safety in ocean environments.

# Features
- Real-time sensor monitoring
- Emergency alert system
- Live data visualization dashboard
- Location tracking system

# Tech Stack
- HTML, CSS, JavaScript (For website),Java (for Application)
- Firebase (for real-time data)
- IoT Sensors

## 📌 Purpose
This project aims to detect emergency conditions in marine environments and provide instant alerts to improve response time and safety.
